AgentBreak

Chaos proxy for testing how your agents handle failures. Supports OpenAI, Anthropic, and MCP.

Quick Start

pip install agentbreak
agentbreak init
agentbreak serve

Endpoints:

  • POST /v1/chat/completions (OpenAI)
  • POST /v1/messages (Anthropic)
  • POST /mcp

MCP Inspect

agentbreak inspect
agentbreak serve

Discovers upstream MCP tools and writes .agentbreak/registry.json.

Current Scope

  • Targets: llm_chat, mcp_tool
  • MCP methods: tools/list, tools/call, resources/list, resources/read, prompts/list, prompts/get
  • Faults: http_error, latency, timeout, empty_response, invalid_json, schema_violation, wrong_content, large_response

Verification

agentbreak verify
